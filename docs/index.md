# Provider-V2

统一 AI 网关，插件化生态 **Provider-Evo**。

## 当前版本

**2.2.0-alpha**

## 插件类型

| 类型 | 说明 |
|------|------|
| platform | 平台适配器（如 qwen、zen） |
| fncall | 工具协议扩展 |
| webui | WebUI 增强 |
| coplan | CodingPlan / 策略组（entropy） |

## 快速开始

```bash
pip install provider-guide==2.2.0-alpha
python main.py
```

WebUI 默认端口 `1337`；插件管理在侧栏「插件」页。
